# 安装入口补充 AI 治理快速检查需求

**日期**: 2026-03-26  
**目标**: 在公开安装入口与安装提示词中补充“AI 治理是否配置好”的快速检查命令与提示词约束。

## 需求

1. 在公开安装入口中补充一个很短的“安装后快速检查 AI 治理配置”说明
2. 在安装提示词中要求安装助手在安装完成后主动给出快速检查命令
3. 命令必须能明确指向目标宿主根目录，而不是模糊依赖默认值
4. 不新增新的公开入口文档，不制造文档扩散

## 验收标准

1. 中文与英文入口页都出现快速检查说明
2. `full` 与 `minimal` 两份安装提示词都要求安装完成后给出快速检查命令
3. 文案区分“本地安装完成”和“AI 治理在线 advice 是否就绪”
