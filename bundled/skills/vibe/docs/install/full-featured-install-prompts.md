# Codex 深度安装路径说明（兼容页）

如果你现在要安装或更新，请先回到唯一公开入口：

## 这页现在保留什么

- 说明这是一份旧的 Codex 深度安装提示词入口兼容页
- 告诉读者当前公开安装面已经收束到一个入口
- 为需要研究命令边界或宿主细节的用户提供继续阅读的参考

## 什么时候还需要看这页

- 你在追踪旧文档、旧链接或历史讨论里的 “Codex 深度安装提示词”
- 你已经知道自己在看 Codex 宿主的补充说明，而不是在找公开安装入口
- 你想进一步看多宿主命令参考或宿主边界说明

## 当前建议

- 普通安装与更新：使用 [`one-click-install-release-copy.md`](./one-click-install-release-copy.md)
- 需要多宿主命令边界：使用 [`recommended-full-path.md`](./recommended-full-path.md)
- 需要看宿主本地配置与边界：使用 [`host-plugin-policy.md`](./host-plugin-policy.md)

## 关于补充配置

- 基础安装完成后即可直接使用
- 如果还想补在线 provider、MCP、CLI 或宿主本地 settings，这些都应理解为增强建议，而不是基础安装失败
- 各项补充能力是否仍由宿主侧本地维护，应以当前公开提示词和参考文档中的 truth-first 说明为准
