# 中文 README 与英文新版结构对齐需求文档

## 目标

将中文 `README.zh.md` 的公开叙事结构收敛到当前英文新版 `README.md` 的主线口径，避免中文页仍保留旧安装入口表述和旧路由章节结构。

## 交付物

- 更新 [`README.zh.md`](../../../README.zh.md)

## 约束

- 保持中文 README 的自然中文表达，不做机械直译
- 结构与产品口径需要对齐英文新版
- 不引入绝对路径或环境相关分发信息

## 验收标准

1. 中文 README 在路由章节主线中补齐“Why this design?”对应内容。
2. 中文 README 的路由 FAQ 不再重复承载这一主线说明。
3. 中文 README 安装章节改为“一个入口，两种公开版本”的新口径。
4. 中文 README 的安装表格与英文新版表达的产品结构一致。
